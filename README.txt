== Installation ==

<h2>This section describes how to install the plugin and get it working.</h2>

e.g.

<h4>First things first, thank you for choosing our Advanced Customer Account plugin!</h4>
<h4>You can install the plugin one of two ways:</h4>
<h4>Install using FTP</h4>
<ul>
	<li>Unzip the <strong>advanced-customer-account.zip</strong> file locally to your machine</li>
	<li>Connect to your FTP area using your preferred FTP package</li>
	<li>Upload the <strong>Xadvanced-customer-account.zip</strong> folder that you extracted from the zip file to the plugin folder of your WordPress installation (<strong>wp-content -&gt; plugin</strong>)
	</li>
	<li>Go to <strong>plugin </strong> and activate Advanced Customer Account.</li>
</ul>
<h4>Install by WordPress (recommended)</h4>
<ul>
	<li>Unzip the <strong>advanced-customer-account.zip</strong> file locally to your machine</li>
	<li>Connect to your FTP area using your preferred FTP package</li>
	<li>Upload the <strong>advanced-customer-account.zip</strong> folder that you extracted from the zip file to the plugin folder of your WordPress
	installation (<strong>wp-content -&gt; plugin</strong>)
	</li>
	<li>Go to <strong>plugin </strong> and activate Advanced Customer Account.<br>
	</li>
</ul>

== Frequently Asked Questions  ==

= Can I add new Features in customer account page   =

 yes, you can add new Features by this plugin

= Can I see last few order in account dashboard page =

 Yes, You can show last 3 orders in account dashboard tab.
 
= Can I use the plugin without woocommece plugin =

 No, The plugin is addon for woocommece so don't use without woocommece plugin.
